How to Configure SAML 2.0 for Clari

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Copy and save the following IdP Metadata show below with the filename metadata.xml. 
    Sign into the Okta Admin dashboard to generate this value.
  2. Download following x.509 certificate:

    Sign in to the Okta Admin app to generate this link.

  3. Send an email message to Clari and request that they enable SAML for your organization.

  4. Attach the metadata.xml file you created in step 1 and the x.509 certificate you downloaded in step 2.
  5. Clari will enable SAML and provide you with an Organization ID.
  6. After you receive the confirmation from Clari, in Okta, navigate to the General tab for the Clari app and enter the Organization ID that Clari provided.
  7. Done!

    Note: This application only works on a mobile device through a native SSO environment.
