How to Configure SAML 2.0 for Klipfolio

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Sign into Klipfolio. From the main screen, navigate to Account > Settings > Single Sign-On, as shown below.



  2. Select Edit next to Single Sign-On, as shown above. The following screen appears.

  3. Select SAML 2 in the Single Sign-On (SSO) field, as shown above.
  4. Copy the following Certificate and paste it into the Certificate field, as shown above.

    Sign in to the Okta Admin app to generate this variable.

  5. Select Save.

  6. Done!
Note: IdP-initiated flows are supported. SP-initiated flows and Just In Time (JIT) provisioning are not supported.
