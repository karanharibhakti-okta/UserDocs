How to Configure SAML 2.0 for Shareworks

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Send an email message to Shareworks and request that they enable SAML for your organization.
  2. Copy the following Issuer and paste it in the email:

    Sign in to the Okta Admin app to have this variable generated for you.

    Sign into the Okta Admin dashboard to generate this value.
  4. Shareworks will process your request and notify you when SAML is enabled.
  5. After you receive the confirmation email from Shareworks, you can assign people to the application.
  6. Done!
Note: Only an IdP-initiated flow is supported. SP-initiated flows and Just In Time (JIT) provisioning are  not supported.
