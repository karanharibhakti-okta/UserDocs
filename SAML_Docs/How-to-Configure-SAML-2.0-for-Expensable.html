How to Configure SAML 2.0 for Expensable

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

Email a Provisioning Request

 

  1. Send an email to Expensable support (techsupport@expensable.com). In the message,

    • Request SAML 2.0 access for your company.

  2. Attach the following three items to the mail.

    Issuer URI

    Sign into the Okta Admin dashboard to generate this value.

    SSO Service URL

    Sign into the Okta Admin dashboard to generate this value.

    Assertion Signing Certificate

    Sign into the Okta Admin dashboard to generate this value.

  3. Done!
